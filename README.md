# 🚀 Marquee MERN

A modern MERN Stack application built using **MongoDB, Express.js, React.js, and Node.js**. This project demonstrates a responsive full-stack web application with a clean UI and seamless communication between the frontend and backend.

## 📌 Features

- ⚡ MERN Stack Architecture
- 🎨 Responsive User Interface
- 🔄 Dynamic Data Rendering
- 🌐 REST API Integration
- 📦 MongoDB Database
- 🔥 Express & Node.js Backend
- ⚛️ React Frontend
- 📱 Mobile Friendly Design

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/milapbhut/Marquee_mern.git
```

### Navigate to the Project

```bash
cd Marquee_mern
```

### Install Dependencies

#### Backend

```bash
cd server
npm install
```

#### Frontend

```bash
cd ../client
npm install
```

---

## ▶️ Run the Project

### Start Backend

```bash
cd server
npm start
```

### Start Frontend

```bash
cd client
npm start
```

The application will be available at:

- Frontend: `http://localhost:3000`
- Backend: `http://localhost:5000`
